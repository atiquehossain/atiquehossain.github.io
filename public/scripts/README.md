# Script Guide

This folder is the readable script layer used by `index.html`.

Main entries:

- `app-main.js` -> app entrypoint
- `app-core.js` -> main bundled runtime
- `../styles/main.css` -> stylesheet entrypoint

Readable modules:

- `locale.js`
- `project-list.js`
- `project-thumbnails-map.js`
